# Multi-Paradigm Languages

A language X is said to support paradigm Y if the language offers the possibility to program in the style defined by the paradigm.
Many languages are multi-paradigm.

It pretty much boils down to machine code, since regardless of the programming language, all the CPU can understand is machine code which is the binary representation of assembly.

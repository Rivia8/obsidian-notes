# Introduction

COMP26020 looks at a range of programming paradigms and languages.

- 30:70 Coursework to Exam
- Imperative Programming in C,
- OOP in C++
- Functional Programming in Haskell,
- Concurrent Programming and resource management in Rust.

Course unit uses a blended approach.
There are weekly videos, quizzes and formative programming exercises and weekly lectures.

Lectures will have information not present in the 'home' material.

At the end of the year there will be an online exam which consists of a set of auto-marked questions.

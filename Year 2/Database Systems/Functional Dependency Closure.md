# Functional Dependency Closure

$F^+$ is all the [[Functional Dependencies|FDs]] that can be derived, given a set of FDs F.
By determining the closure of $F$, we can design normalised databases.


![[Pasted image 20251020142941.png#invert|600]]
![[Pasted image 20251020143031.png#invert|800]]


You can almost never calculate the *entire* closure, but the idea of it is critical for database [[Normalisation|normalisation.

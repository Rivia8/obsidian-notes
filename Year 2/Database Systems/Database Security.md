# Database Security

This is the practise of securing the data held in databases.

There are three main commonly accepted security goals in databases and we need to protect from the loss of them:

1. Loss of integrity - prevention from improper modification,
2. Loss of availability - losing the ability to access the database,
3. Loss of confidentiality  - preventing the database from unauthorised disclosure.

DB security is incredibly important since an organisation's reputation, financial status, legal compliance and general functionality is at stake.

Common DB Security Threats:
- Insider Threats,
- Buffer Overflow Exploitations,
- Backup Attacks,
- Database Software Vulnerabilities,
- Denial of Service Attacks,
- Malware,
- SQL/NoSQL injection Attacks.

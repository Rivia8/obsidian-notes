# Operating System Kernel

Because of the close proximity with the hardware and performance, C is heavily used in OS kernels.
There are two big OS operations that are implemented in C: 

- [[Context Switching]]
- [[System Calls]]

Here we will look at these two things within the Linux kernel code since Linux is used absolutely everywhere - in Android phones and most embedded or loT systems.

# Sequential Logic

Sequential logic describes circuit whose output depends on the current inputs and its previous state (what it has stored in memory), the way this works is that the logic is always active and it changes depending on the **clock**.

What is the clock?
- The clock is a component that emits a signal that synchronised components and components will update at different points along the clocks wave i.e. the positive edge of the clock (when the clock signal rises) or the negative edge of the clock (where the clock signal falls).

Usually with sequential logic we design circuits like flip-flops and registers.

# The D-Type Flip-Flop

**Flip-Flops:**

A sequential system needs a memory element to store information, this is implemented practically using a device called a flip-flop.
Activation of inputs causes the flip-flop to assume one of two states. '0' and '1', which is remembered.
It remains in this state until driven into another state by suitable activation of one or more inputs.
Hence, the flip-flop is a simple one bit memory element, you can store either a '0' or '1' to the device, which it will remember until the value is overwritten.

The D-type FF is an edge triggered device as a minimum it will have two inputs:
- A data input 
- A clock input

**State**:
What does the value stored in the memory in a sequential system represent?: It defines the state of the sequential circuit at that time.
The next state of the system is determined by its current state and the state of any input signals.
Each possible state is identified by a unique number, determined by the range of values that can be stored to the memory element.

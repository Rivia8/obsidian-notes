# Regression

In [[Machine Learning|machine learning]] we have learn tat regression tries to associate the input with continuous outputs.

## Linear Regression

This is a type of **supervised** learning algorithm, used for regression tasks.

The idea is if you have a bunch of data points plotted on a graph, linear regression tries to find the "best-fit" straight line that represents the relationship between the input features and the output variable.

The loss function helps us measure how far off our model's prediction are from the actual values and the goal is to minimise the loss.


## Polynomial Regression

Polynomial regression is essentially linear regression but allows it to fit **curves** rather than just straight lines.
This is used when you believe that the relationship between the input and the output is not a simple straight line.


## Logistic Regression

This is similar to linear regression but instead of learning labels to output a regression based value (continuous outputs) it predicts the **probability** that an output belongs to a specific category or class.

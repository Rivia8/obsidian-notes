# Partial Functions

A partial function, unlike a regular function, is only defined for some elements of its domain.

![[Pasted image 20250224011204.png#invert]]

if $f$ is a partial function from $A$ to $B$, we write: $$f: ~ A \rightharpoonup B$$
*(we use the half-tip arrow for partial function)*

Let's say we have two sets, 
- $A=\{1, 2, 3, 4\}$
- $B=\{a,b,c\}$ 

A partial function might be defined as:
$f(1) = a$  and $f(3) = b$.

## The Domain of Definition

The domain of definition is the set of inputs for which the function is defined - its the subset of the original domain where the function provides an output.

The **domain of definition** of a partial function $f: S \rightharpoonup T$ is defined as: $$ dom ~ f ~ = \{s~~\in~~S~~|~~f\,s~~~~is~~~defined\}$$

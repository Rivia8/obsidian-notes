# Linear Algebra

**Linear Algebra** is a branch of mathematics that deals with vectors, matrices and linear transformations.

At its core it is about understanding **linear relationships** between mathematical objects. These include:

- [[Vector Spaces|Vectors]]
- Matrices
- Scalars

Linear algebra is a foundational tool in many areas like computer graphics, machine learning and engineering.

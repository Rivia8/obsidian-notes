# Properties of Functions
Functions allow us to transport elements from one set to another.
## Injectivity

![[Pasted image 20241012134806.png#invert]]

Injective (one-to-one) functions are that of functions that for every element in the domain is mapped to distinct element in the codomain. This means that if two different inputs are given, they must lead to different outputs.

This also means that if the function has the same output for two different inputs, the inputs must be equal to each other.

## Bijectivity
A function that is both injective and surjective is called bijective. 
This means that every element in the domain maps to a unique element in the codomain (domain being all the the possible valid inputs).

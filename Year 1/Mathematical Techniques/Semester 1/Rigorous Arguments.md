# Rigorous Arguments
## Binary Operations
Binary operation is the process of involving two inputs and having an output.

**Commutativity**:
![[Pasted image 20241005195203.png#invert]]
This means that if you were to swap the first or second element, it would result in the same output.

**Associativity**:
![[Pasted image 20241005211051.png#invert]]
When calculating with such a binary operation we may change the bracketing without affecting the result.

**Unit**:
![[Pasted image 20241005212310.png#invert]]
A unit is sometimes called a neutral element, this means that when you apply the operation to it and another input returns that other input.

**Inverse:**
![[Pasted image 20241005223525.png#invert]]
Applying the operation to an inverse and some other element as undoing the effect of applying the operation to the given element.

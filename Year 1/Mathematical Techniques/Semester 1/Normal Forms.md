# Normal Forms

## Conjunctive Normal Forms
This is a logical formula is in CNF if it is a conjunction (AND) of one or more clauses, where each clause is a disjunction (OR)
![[Pasted image 20241111091844.png#invert]]

This is a form where all the terms is separated by ANDs and expanding in each term, the sub terms are separated by ORs:
![[Pasted image 20241111101007.png#invert]]

## Disjunctive Normal Forms
This is a logical formula where is it in DNF if it is a disjunction (OR) of one or more clauses where each clause is a conjunction (AND).
![[Pasted image 20241111094415.png#invert]]
Disjunctive normal forms have the same properties as conjunctive normal forms; only the role of ∧ and ∨ are reversed
